How to Generate a PGP Key

Last Updated: [Insert Date]

DarkVault uses PGP encryption for secure communication. You’ll need a PGP public key to register. Follow these steps to generate one using GPG (GnuPG), a free and widely-used tool.

Step 1: Install GPG

Download and install GPG for your system:

  • Windows: Get Gpg4win from gpg4win.org.
  • macOS: Install via Homebrew (brew install gnupg) or from gnupg.org.
  • Linux: Install via package manager (e.g., sudo apt install gnupg on Ubuntu).

Step 2: Generate a Key Pair

Open a terminal/command prompt and run:

gpg --gen-key

Follow the prompts:

  • Select “RSA and RSA” (default) and press Enter.
  • Choose a key size (e.g., 2048 or 4096 bits).
  • Set an expiration (e.g., “0” for no expiry).
  • Enter a name (can be a pseudonym), optional email, and a strong passphrase.

Step 3: Export Your Public Key

List your keys to find your key ID:

gpg --list-keys

Export your public key in text format:

gpg --armor --export your_email@example.com > mypublickey.asc

Replace your_email@example.com with the email or ID you used.

Step 4: Copy Your Public Key

Open the file (mypublickey.asc) in a text editor, or display it: 

cat mypublickey.asc

Copy the entire block, including -----BEGIN PGP PUBLIC KEY BLOCK----- and -----END PGP PUBLIC KEY BLOCK-----, and paste it into the registration form.

Step 5: Secure Your Private Key

Never share your private key or passphrase. Store them offline or in a secure location. You’ll need them to decrypt messages from DarkVault.

Tips

  • Use a strong passphrase to protect your private key.
  • Backup your key pair in a safe place.
  • For GUI options, try Kleopatra (Windows) or OpenKeychain (Android).
